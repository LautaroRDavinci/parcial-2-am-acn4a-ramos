# fit-routine

Una app para Android donde podés anotar tus rutinas de ejercicio del día.

proyecto para la materia de desarrollo mobile. Todavía es básica pero la idea es que puedas escribir los ejercicios que vas a hacer y agregarlos a una lista.

## ¿Qué hace?

- Tiene una pantalla principal con un campo de texto
- Escribís el ejercicio y le das al botón para agregarlo
